## 资源卡片详情

每个资源卡片包含：
- **图标**：60x60像素，支持多种格式
- **名称**：加粗显示，单行溢出省略
- **版本**：灰色小字，格式为 "v版本号"
- **简介**：最多2行，溢出省略
- **标签**：最多显示3个，圆角背景
- **按钮**：查看详情按钮

## 交互流程

### 浏览资源
1. 用户点击导航栏的"商店"图标
2. 显示商店页面，默认显示"主题"标签
3. 自动加载资源列表（显示加载指示器）
4. 资源以网格方式展示

### 查看详情
1. 用户点击资源卡片的"查看详情"按钮
2. 弹出详情对话框
3. 显示完整的资源信息
4. 提供外部链接（官网、仓库等）

### 安装资源
1. 用户在详情对话框点击"安装"按钮
2. 显示"开始下载..."提示
3. 异步下载资源文件
4. 下载完成后：
   - 壁纸源：自动导入，显示"安装成功"
   - 主题/插件：显示"下载完成"（安装待实现）

### 切换标签
1. 用户点击标签（主题/壁纸源/插件）
2. 清空当前列表
3. 显示加载指示器
4. 加载新标签的资源

### 配置商店源
1. 用户进入"设置" → "资源"
2. 找到"商店源"配置区域
3. 启用"使用自定义源"开关
4. 输入自定义URL
5. 下次打开商店时生效

## 视觉设计

### 颜色方案
- **卡片背景**：Surface
- **按钮**：Primary/Secondary
- **文字**：默认/Grey
- **错误提示**：Error

### 间距
- **卡片间距**：16px
- **内边距**：16px
- **组件间距**：12px

### 字体大小
- **页面标题**：24px
- **卡片标题**：16px
- **版本号**：12px
- **简介**：12px
- **标签**：10px

### 交互反馈
- **悬停**：卡片阴影增强
- **点击**：按钮涟漪效果
- **加载**：进度环动画
- **成功**：绿色SnackBar
- **失败**：红色SnackBar

## 响应式设计

- **桌面**：3列网格（每列最大350px）
- **平板**：2列网格
- **手机**：1列网格

网格会根据窗口大小自动调整列数。

## 无障碍特性

- 所有按钮都有工具提示
- 图标提供备用文本
- 键盘导航支持
- 颜色对比度符合WCAG标准
